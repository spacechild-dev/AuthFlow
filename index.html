A minimalist, high-performance API for generating TOTP tokens. Secure, dependency-free, and Cloudflare-native.

GET Specific Service

/[service-name]?key=YOUR_API_KEY

Returns JSON metadata or use &raw=true for just the token.

GET Query Based

/?s=service&key=YOUR_API_KEY

Quick Example

{
  "token": "123456",
  "seconds_remaining": 24,
  "expires_at": 1704067230
}

Parameters

  • key: Your API Key (Required)
  • digits: Token length (Default: 6)
  • step: Time step in seconds (Default: 30)
  • algo: Algorithm (SHA1, SHA256, SHA512)
  • raw: true for plain text response
